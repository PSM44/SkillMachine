WBS — SKILL CAE.C1.EXAM v1.0

Nombre: CAE.OPERATOR
Propósito: Aprobar Cambridge C1 Advanced (CAE) con nivel C o superior
Modo: ejecución + evaluación continua (no estudio pasivo)

00.00 PRINCIPIO RECTOR
El examen mide performance bajo presión, no conocimiento teórico
La preparación es simulación + corrección + iteración
Writing y Speaking definen el resultado final (alto impacto)
01.00 INPUT CONTRACT
01.01 Objetivo
aprobar CAE ≥ nivel C
score target: 180+
01.02 Restricciones
tiempo: 60–90 min diarios
duración: 6–8 semanas
herramientas: GPT + material oficial
01.03 Nivel inicial (autoevaluación)
Writing: bajo / medio / alto
Speaking: bajo / medio / alto
Grammar: bajo / medio / alto

👉 Si no haces este diagnóstico, el plan queda mal calibrado.

02.00 MAPA DEL EXAMEN
02.01 Módulos
M1 Reading (L3)
M2 Use of English (L3)
M3 Writing (L4)
M4 Listening (L3)
M5 Speaking (L4)
02.02 Priorización real
CORE: Writing + Use of English
SOPORTE: Reading + Listening
DIFERENCIAL: Speaking

👉 Error común: estudiar todo igual → bajo rendimiento.

03.00 CICLO OPERATIVO (DIARIO)

Duración: 60–90 min

03.01 BLOQUE A — USE OF ENGLISH (15–20 min)

Objetivo: precisión gramatical

Prompt GPT

“Generate 10 CAE Use of English questions (mixed parts 1–4). Include answers separately.”

Ejecución
responder sin ayuda
revisar respuestas
Feedback

“Explain only my incorrect answers and patterns of error.”

03.02 BLOQUE B — WRITING (20–25 min)

Objetivo: scoring alto

Prompt GPT

“Give me a CAE Writing task (Part 1 or 2). After I answer, correct it strictly using Cambridge criteria: content, communicative achievement, organization, language. Assign a band score.”

Output obligatorio
texto corregido
lista de errores
versión mejorada
03.03 BLOQUE C — SPEAKING (10–15 min)

Objetivo: fluidez + coherencia

Prompt GPT

“Act as a CAE examiner. Run Speaking Part 2. Ask me to compare two situations. Then give feedback on fluency, grammar, and vocabulary.”

Ejecución
hablar en voz alta (no mental)
grabarse (ideal)
03.04 BLOQUE D — INPUT CONTROLADO (10–15 min)

Objetivo: vocabulario + comprensión

Acción
leer/listening material C1
Prompt GPT

“Extract advanced vocabulary and typical CAE expressions from this text, with examples.”

03.05 BLOQUE E — ERROR LOG (5 min)

Archivo obligatorio:

/CAE_C1/04_errors.txt
- error
- tipo (grammar / vocab / structure)
- corrección
- ejemplo correcto
