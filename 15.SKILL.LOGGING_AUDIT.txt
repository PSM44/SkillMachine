==========
01.00_DEFINICIONES
==========

01.10_TASK_ID
- Identificador único por “intención”.
Ejemplo: TASK-20260226-1420-PROJ0001-0007

01.20_RUN_ID
- Identificador único por ejecución concreta (reintentos cuentan).
Ejemplo: RUN-20260226-1421-0001

01.30_PROJ_ID
- ID del proyecto (compatible con WBS/ID).

01.40_CHECKPOINT_ID
- Git commit SHA (preferido) o SNAP-YYYYMMDD-HHMM-SEQ (fallback).

==========
02.00_CARPETA_Y_NAMING_DE_LOGS
==========

02.10_ROOT
- \03_ARTIFACTS\LOGS\

02.20_SUBCARPETAS_MINIMAS
- \LOGS\RUNS\        (1 archivo por RUN_ID)
- \LOGS\DAILY\       (agregados diarios)

02.30_NAMING
- RUN: LOG.RUN.<RUN_ID>.txt
- DAILY: LOG.DAILY.YYYYMMDD.ACTIVE.txt
- MONTHLY: LOG.MONTHLY.YYYYMM.ACTIVE.txt

==========
03.00_FORMATO_DE_LOG_TXT
==========

03.10_BLOQUE_EVENTO
Cada evento:
- TIMESTAMP
- LEVEL
- PROJ_ID
- TASK_ID
- RUN_ID
- STAGE (plan/apply/validate/rollback)
- ACTION (create/edit/move/quarantine/script/model_call)
- PATHS (si aplica)
- RESULT (OK/FAIL)
- COST (si aplica)

03.20_EJEMPLO_MINIMO
TIMESTAMP.........: 2026-02-26 14:21:33
LEVEL.............: INFO
PROJ_ID...........: PROJ_0001
TASK_ID...........: TASK-20260226-1420-PROJ0001-0007
RUN_ID............: RUN-20260226-1421-0001
STAGE.............: APPLY
ACTION............: PATCH_APPLIED
PATHS.............: \02_CODE\src\core.ts
RESULT............: OK
CHECKPOINT_ID.....: a1b2c3d (git)
NOTES.............: Patch aplicado; lint pendiente.

==========
05.00_NIVELES_Y_SEVERIDAD
==========

- INFO: normal
- WARN: anomalía recuperable
- ERROR: FAIL
- Si ERROR por boundary o secrets => elevar a TROUBLE/INCIDENT según skill.

==========
06.00_EVIDENCIA_MINIMA
==========

- Lista de archivos tocados
- Checkpoint_id
- Si hubo script: comando + output
- Si hubo modelo: tokens + costo

==========
07.00_RETENCION_Y_ROTACION
==========

- RUN logs: mantener 60 días (default)
- DAILY: mantener 90 días
- MONTHLY: mantener 24 meses
- SQLite: compact mensual

==========
08.00_EXPORTES_Y_REPORTES

- Reporte diario: top tasks por costo
- Reporte mensual: costo por proyecto + por tipo de tarea

09.00_ANTIPATRONES
